(inner) find(table, query, options, cb)

Query a table using query object. You can call any of the other functions using options.queryType. If not specified, defaults to 'all' type search.
Parameters:
Name Type Description
table string The name of the table to be queried
query Query | Array.<Query> An array composed of objects with `key: value` pairs to be matched. An empty object `{}` results in the whole table being returned.
options Object Search options
Properties
Name Type Description
n number Only for *N queries, the number of results to return
queryType string The type of query to perform. One of: id, anyN, anyOne, all, one, n, any.
cb function Callback function (error?, resultsObject)
